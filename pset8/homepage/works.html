<h1>headings<h1>

Headings are like titles, but they’re actually displayed on the page. HTML provides six levels of headings, and the corresponding elements are: <h1>, <h2>, <h3>, … , <h6>. The higher the number, the less prominent the heading.

<h2>emphasis (italic) elements<h2>

The part wrapped in <em> tags should render as italics. <p> is a block-level element, while <em> is an inline element that affects a span of text inside of a paragraph. It stands for “emphasis”, and it’s typically displayed as italicized text.

<h3>strong (bold) elements<h3>

If you want to be more emphatic than an <em> tag, you can use <strong>. It’s an inline element just like <em>

To draw even more attention your a span of text, you can nest a <strong> element in an <em> element (or vice versa). This will give you text that is both strong and emphasized.

<h4>line breaks<h4>

To tell the browser that we want a hard line break, we need to use an explicit <br/> element.

The <br/> element is useful anywhere text formatting matters. <br/>
Haiku, music lyrics, and signatures are just a few examples where it might come in handy.
